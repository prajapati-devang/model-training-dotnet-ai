# üß† .NET AI Sentiment Analyzer (.NET 10 & Blazor Server)

This project demonstrates a complete end-to-end implementation of a custom Machine Learning model built with **ML.NET** and integrated into a **Blazor Server** web application using **.NET 10**.

## üöÄ Features
- **Custom AI Model:** A binary classification model trained to detect sentiment (Positive/Negative).
- **High Performance:** Utilizes `.NET 10` optimizations and `PredictionEnginePool` for thread-safe, scalable web inference.
- **Real-time UI:** A reactive Blazor interface that provides instant feedback and confidence scores.

---

## üìÇ Project Structure

- **`ModelTrainer/`**: A C# Console Application used to train, evaluate, and export the ML model.
- **`SentimentWebApp/`**: A Blazor Server application that consumes the trained model.
- **`SentimentModel.zip`**: The serialized AI model (generated by the trainer).
